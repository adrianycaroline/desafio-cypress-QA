# Desafio Cypress QA

Este projeto foi desenvolvido como parte de um desafio de automação de testes utilizando o **Cypress**, com o objetivo de validar fluxos críticos de uma loja virtual.

---

## Instalação do Projeto

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/adrianycaroline/desafio-cypress-QA.git
2. Acesse o diretório do projeto:
   ```bash
    cd desafio-cypress-QA
3. Instale as dependências:
   ```bash
   npm install
   npx cypress open
Cenários Automatizados
1. Login com sucesso e falha
2. Adição de produto ao carrinho e finalização de compra

## Motivo da escolha
Fluxo de compra é reutilizado em múltiplos testes e cobre funcionalidades essenciais (login, produto e checkout).
